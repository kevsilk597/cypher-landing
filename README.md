# Cypher Landing

Marketing and product narrative site for Cypher, built to communicate the product vision and workflow value clearly to non-technical and technical audiences.

## Purpose

- Explain the Cypher problem space and product thesis.
- Present core workflow concepts and product positioning.
- Provide a clean external entry point separate from the main build repo.

## Stack

- Next.js
- React
- TypeScript
- Framer Motion

## Local development

```bash
npm install
npm run dev
```

## Notes

- This repository is intentionally focused on the landing experience.
- Core product implementation and technical workflow logic live in the main `cipher` repository.
